## About Web Bimbingan Karir 2025 (Ticketing App)

## About Ticketing App (Web Bimbingan Karir 2025)

Ticketing App is a modern web application designed to simplify the process of discovering, booking, and managing event tickets for users and organizers. Built for the 2025 Bimbingan Karir program, this platform provides a seamless experience for:

- Browsing various events (concerts, seminars, workshops, exhibitions)
- Filtering events by category
- Viewing event details, locations, and prices
- Securely purchasing tickets online
- Managing orders and user profiles

### Key Features

- User-friendly dashboard for event discovery
- Category-based filtering for quick navigation
- Detailed event cards with images, dates, locations, and prices
- Secure order and payment system
- Admin panel for event and ticket management
- Responsive design for desktop and mobile

### Technologies Used

- Laravel (PHP framework)
- Blade templating
- Tailwind CSS for modern UI
- Vite for asset bundling
- MySQL for database

### Getting Started

1. Clone this repository
2. Copy `.env.example` to be `.env`
3. Set up your `.env` file and database
4. Installing dependencies vendor with `composer install`
5. Don't forget `php artisan key:generate`
6. Symlink the storage of asset `php artisan storage:link`
7. Installing dependencies node_module with `npm install`
8. Building the frontend module `npm run build`
9. Migrate database with ORM & seeding all data using `php artisan migrate --seed`
11. Running simultaneously (Frontend & Backend) local `composer run dev`
12. Or running local with `npm run dev` & `php artisan serve`
